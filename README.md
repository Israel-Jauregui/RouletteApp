Roulette Wheel:
The wheel is based on the American Roulette wheel (0 - 37), 0 counts as a house win.

Spin the wheel using the flick gesture.

Each spin will cost $25.

The wheel will stop on a randomly generated number after 4 seconds.

Allows the user to place the following bets: Odds/Evens, Red/Black, Ranges (1-9, 10-19, 20-29. 30-37), or a specific number.

If the user is out of money, more can be aquired by clicking the "$" button which allow the user to click a dollar ($1 each click).

The game has a solid state, so it will save your amount of money when you close the app.
